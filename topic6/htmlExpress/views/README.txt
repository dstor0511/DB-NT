# Project: HTML Express

## Description
HTML Express is a web application project designed to demonstrate the use of Express.js for building server-side applications. The project showcases how to serve HTML content, handle routing, and manage middleware in an Express.js environment.

## Features
- Serve static HTML files
- Dynamic routing
- Middleware integration
- Error handling

## Installation
1. Clone the repository:
    ```
    git clone <repository-url>
    ```
2. Navigate to the project directory:
    ```
    cd topic6/htmlExpress
    ```
3. Install the dependencies:
    ```
    npm install
    ```

## Usage
1. Start the server:
    ```
    npm start
    ```
2. Open your web browser and navigate to `http://localhost:3000` to view the application.

## Project Structure
- `views/`: Contains HTML files to be served
- `routes/`: Defines the routing logic
- `public/`: Contains static assets like CSS and JavaScript files
- `app.js`: Main application file
